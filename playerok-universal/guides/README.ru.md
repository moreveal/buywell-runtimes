# Playerok Universal

Модуль связывает установленный Playerok Universal со сценариями Buywell.

- Запускает сценарий при новой оплаченной продаже.
- Запускает сценарий при новом сообщении клиента.
- Показывает категории и матрицу товаров продавца в подключении сценария.
- Позволяет фильтровать сообщения по тексту, имени отправителя, товару, игре и категории.
- Позволяет фильтровать покупки по товару, цене, имени покупателя, игре, категории и способу получения.
- Оставляет фильтры по ID для точных интеграций и списков.
- Выполняет действие **Отправить сообщение** в чате исходного события.
- Хранит очередь доставки и защиту от повторов в локальной SQLite-базе.

Выберите категорию, чтобы загрузить все свои товары в ней. Поле **Товар Playerok** позволяет независимо сопоставить каждый товар со входами сценария: сроком, количеством, тарифом или другим нормализованным значением. В той же матрице доступны способ получения, характеристики категории и поля заказа.

Новый товар или вариант, который ещё не сопоставлен и не исключён, не запускает сценарий с неверными данными: Buywell попросит обновить матрицу.

При первом подключении старые продажи и сообщения не запускают сценарии.

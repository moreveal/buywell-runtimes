# Установка GGSel Seller

Нужны Python 3.11+, API-ключ продавца GGSel с доступом к Orders и Chats V1 и ключ подключения Buywell. Публичный IP и входящие порты не нужны.

## 1. Скачайте и распакуйте

Скачайте runtime ZIP из установленного модуля GGSel Seller `1.1.0` в Buywell и распакуйте его в постоянную папку. После установки папку лучше не переносить.

## 2. Запустите установщик

Windows — дважды нажмите `install.bat`.

Linux:

```bash
chmod +x install.sh
./install.sh
```

Установщик сам:

- найдёт Python и попробует восстановить отсутствующий `pip` через `ensurepip`;
- создаст отдельное окружение и установит зависимости;
- запросит ключ Buywell, ID продавца и API-ключ GGSel;
- проверит доступ к покупкам и чатам;
- выполнит настоящий handshake runtime с Buywell;
- предложит установить и сразу запустить фоновую службу для текущего пользователя.

На Linux служба создаётся в systemd с текущим пользователем и текущей папкой. На Windows создаётся задача `Buywell GGSel Runtime`, которая запускается при входе текущего пользователя.

## 3. Проверьте запуск

Linux:

```bash
sudo systemctl status buywell-ggsel
sudo journalctl -u buywell-ggsel -f
```

Windows: откройте «Планировщик заданий» и найдите `Buywell GGSel Runtime`.

В конце успешной установки должны быть подтверждены три проверки: конфигурация,
GGSel API и подключение Buywell. Повторный запуск установщика обновляет существующую
службу или задачу и перезапускает её, не создавая копию.

После подключения включите нужные событийные подключения в Buywell. Первый запуск запоминает уже существующие покупки и сообщения, но не запускает по ним сценарии.

## Если автоматический запуск не нужен

Запускайте runtime вручную через `run.bat` на Windows или `./run.sh` на Linux.

Если Linux сообщает об отсутствии поддержки виртуальных окружений, выполните `sudo apt install python3-venv` и повторите установку. Ошибка GGSel `403` означает, что ключ не имеет доступа к Orders и Chats V1.

`config.json` и папка `state` содержат секреты и рабочее состояние. Не публикуйте их и переносите вместе с runtime.

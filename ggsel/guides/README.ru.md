# GGSel Seller

Модуль связывает аккаунт продавца GGSel со сценариями Buywell.

Доступные события:

- новая покупка;
- новое сообщение покупателя.

Доступное действие — отправка сообщения в чат текущей покупки. Runtime работает
через исходящее подключение, хранит неподтверждённые события в SQLite и повторяет
доставку после восстановления связи.

GGSel предоставляет список последних продаж без курсора. Runtime опрашивает
максимум `sales_window` записей за один проход. Уменьшайте интервал опроса при
высоком количестве продаж.

# Installing FunPay Cardinal

1. Download this version's runtime from Buywell.
2. Put `buywell_runtime.py` in the `plugins` directory of FunPay Cardinal 0.1.17.8.
3. Restart Cardinal.
4. Open the Buywell plugin settings in Cardinal's Telegram panel.
5. Create a module connection key in Buywell.
6. In Cardinal, choose “Enter connection key” and send the copied key.
7. Return to Buywell and make sure Cardinal is shown as connected.

On its first start, the plugin may install `websocket-client` in Cardinal's Python environment.
